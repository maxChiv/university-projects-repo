  1. Setup Backend files in Docker

docker compose up backend

  1. Check logs 

docker compose logs -f backend

Should see Started Application in X seconds

Creation of Mock Data

Create User (POST /api/v1/users)

curl -X POST http://localhost:8080/api/v1/users \
  -H "Content-Type: application/json" \
  -d '{
        "id": "user-001",
        "name": "Dr. Alice Nguyen",
        "email": "alice@unimelb.edu.au",
        "isAdmin": false,
        "isApprover": true
      }'

Create Project (POST /api/v1/projects)

curl -X POST http://localhost:8080/api/v1/projects \
  -H "Content-Type: application/json" \
  -H "X-User-Id: user-001" \
  -d '{
        "projectId": "P-001",
        "title": "Quantum Sensor Research",
        "description": "Developing new superconducting sensors",
        "ownerUserId": "user-001",
        "status": "DRAFT",
        "details": {
          "funder": "ARC",
          "department": "School of Physics",
          "faculty": "Science",
          "scheme": "Discovery Grant",
          "startDate": "2025-01-01",
          "endDate": "2026-12-31",
          "currency": "AUD"
        }
      }'

and expected result will be 

{
  "projectId": "P-001",
  "status": "DRAFT",
  "title": "Quantum Sensor Research"
}

Submit Project for Approval

(POST /api/v1/projects/{projectId}/approvals/submit)

curl -X POST http://localhost:8080/api/v1/projects/P-001/approvals/submit \
  -H "Content-Type: application/json" \
  -H "X-User-Id: user-001" \
  -d '{
        "actorUserId": "user-001",
        "comment": "Submitting project for review"
      }'

and expected result will be 

{
  "projectId": "P-001",
  "status": "SUBMITTED",
  "message": "Project submitted successfully"
}

Approve Project (POST /api/v1/projects/{projectId}/approvals/approve)

curl -X POST http://localhost:8080/api/v1/projects/P-001/approvals/approve \
  -H "Content-Type: application/json" \
  -H "X-User-Id: user-002" \
  -d '{
        "actorUserId": "user-002",
        "comment": "Approved by Faculty Dean"
      }'

and expected result will be 

{
  "projectId": "P-001",
  "status": "APPROVED",
  "comment": "Approved by Faculty Dean"
}

Add Staff Costs (POST /api/v1/projects/{projectId}/staff-costs)

curl -X POST http://localhost:8080/api/v1/projects/P-001/staff-costs \
  -H "Content-Type: application/json" \
  -d '{
        "staffName": "Dr. John Smith",
        "employmentType": "Fixed-term",
        "category": "Academic",
        "classification": "Level B",
        "timeBasis": "FTE",
        "salaryRate": 95000,
        "wageCost": 105000,
        "fullCost": 120000
      }'

and expected result will be 

{
  "staffName": "Dr. John Smith",
  "fullCost": 120000,
  "calculatedMultiplier": "Full cost recovery applied"
}

Retrieve Price Summary (GET /api/v1/projects/{projectId}/price-summary)

curl http://localhost:8080/api/v1/projects/P-001/price-summary

and expected result will be 

{
  "currency": "AUD",
  "totalCost": 150000,
  "indirectCostMultiplier": 1.2,
  "priceToCharge": 180000
}
