# IT Project (COMP30022)

## Course Description

[IT Project, University of Melbourne Handbook](https://handbook.unimelb.edu.au/2026/subjects/comp30022)

This subject is the capstone project for the Informatics and Computing and Software Systems majors. Students work in small teams on a real-world client problem, applying software engineering principles across analysis, design, implementation, testing, and project delivery.

The subject focuses on both technical delivery and professional practice, including project management, design frameworks, technical reviews, testing, ethics, and product evaluation.

## Project Overview

For this subject, the assignment was a full team software project delivered for a real client.

From the project documentation, our team built a web-based **Research Costing and Pricing Tool (RCPT)** for the University of Melbourne’s Research, Innovation and Commercialisation (RIC) unit. The project modernised an existing Excel workflow by introducing:

- structured web-based data entry,
- automated costing and pricing calculations,
- approval workflow and status tracking, and
- export/reporting outputs for operational use.
